---
title: Testing and Benchmarks
description: How to build and run Chatterino7's unit tests and benchmarks using CMake and GoogleTest.
---

Chatterino7 includes a set of unit tests and benchmarks. These are built using CMake with the `-DBUILD_TESTS=On` and `-DBUILD_BENCHMARKS=On` flags respectively.

## Running Tests

<Steps>
  <Step title="Create a build directory for tests">
    ```sh
    mkdir build-tests
    cd build-tests
    ```
  </Step>
  <Step title="Configure with tests enabled">
    ```sh
    cmake -DBUILD_TESTS=On ..
    ```
  </Step>
  <Step title="Build">
    ```sh
    make
    ```
  </Step>
  <Step title="Run the test binary">
    ```sh
    ./bin/chatterino-test
    ```
  </Step>
</Steps>

## Adding a Test

1. Create a new file for the file you're testing. For example, if you're testing `src/providers/emoji/Emojis.cpp`, create `tests/src/Emojis.cpp`.
2. Add the new file to `tests/CMakeLists.txt` in the `test_SOURCES` variable.

See `tests/src/Emojis.cpp` for simple tests to base your own on. Chatterino7 uses [GoogleTest](http://google.github.io/googletest/primer.html).

## Running Benchmarks

<Steps>
  <Step title="Create a build directory for benchmarks">
    ```sh
    mkdir build-benchmarks
    cd build-benchmarks
    ```
  </Step>
  <Step title="Configure with benchmarks enabled">
    ```sh
    cmake -DBUILD_BENCHMARKS=On ..
    ```
  </Step>
  <Step title="Build">
    ```sh
    make
    ```
  </Step>
  <Step title="Run the benchmark binary">
    ```sh
    ./bin/chatterino-benchmark
    ```
  </Step>
</Steps>
