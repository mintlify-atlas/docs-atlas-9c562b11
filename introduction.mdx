# Chatterino7

Chatterino7 is a fork of [Chatterino 2](https://chatterino.com) — a powerful desktop chat client for Twitch streamers and viewers. This fork adds deep integration with [7TV](https://7tv.app), bringing exclusive cosmetic features not available in the upstream client.

## Core Chatterino features

As a fork of Chatterino 2, Chatterino7 inherits all its powerful features:

- **Split layout** — view multiple channels side by side in a flexible split-pane interface
- **Emotes from multiple providers** — BTTV, FFZ, 7TV, and native Twitch emotes
- **Highlights** — highlight messages by keyword, username, or badge
- **Filters** — filter out messages with custom rules
- **Hotkeys** — fully customizable keyboard shortcuts
- **Commands** — create custom chat commands and aliases
- **Logging** — save chat history to local log files
- **Moderator tools** — timeout, ban, and manage users directly from chat
- **Streamer mode** — hide sensitive information when streaming
- **Custom themes** — create and share fully custom color themes
